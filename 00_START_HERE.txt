المصادقة (Authentication):
├─ POST   /api/v1/auth/login                  ← تسجيل الدخول
└─ POST   /api/v1/auth/create-user [ADMIN]    ← إنشاء مستخدم

إدارة المستخدمين (Admin Only):
├─ GET    /api/v1/admin/users                 ← عرض المستخدمين
├─ PUT    /api/v1/admin/users/:userId         ← تحديث مستخدم
├─ DELETE /api/v1/admin/users/:userId         ← حذف مستخدم
└─ GET    /api/v1/admin/audit-logs            ← عرض السجلات

المزامنة (Sync):
├─ POST   /api/v1/sync/push [TOKEN]           ← رفع البيانات
└─ GET    /api/v1/sync/pull [TOKEN]           ← تنزيل البيانات

النظام (System):
└─ GET    /api/v1/status                      ← حالة النظام

   ✓ الحل:
     1. تأكد من تشغيل mongod
     2. تحقق من MONGODB_URI في .env
     3. استخدم MongoDB Atlas بدلاً من المحلي

   شرح: المنفذ 3000 قيد الاستخدام
   ✓ الحل: PORT=3001 npm run dev
