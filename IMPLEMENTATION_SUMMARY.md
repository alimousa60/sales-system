### 🚀 الخطوات التالية

#### للبدء الفوري:
```bash
# 1. تثبيت المكتبات
npm install

# 2. إنشاء ملف البيئة
cp .env.example .env

# 3. تهيئة قاعدة البيانات
node init-db.js

# 4. تشغيل الخادم
npm run dev
```

#### للنشر في الإنتاج:
```bash
# استخدام Docker
docker-compose up -d

# أو النشر على الخوادم
node server.js

# مع PM2 (مدير العمليات)
pm2 start server.js --name "sales-system"
```
